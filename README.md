# Paper Trader

*A web application that allows users to simulate stock trades and track portfolio performance*

## Tech Stack
- Python backend
- Finnhub API for real-time market data
- Flask web framework
- SQLite database
- HTML & CSS frontend

## Features
- User registration, login/logout, authentication
- Buy and sell simulated equities using accurate market pricing
- Portfolio transaction history and accurate performance simulation
- Live price quotes
